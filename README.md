## 🚀 Overview
ShareBooth brings people together from anywhere — create a booth session, share a 4‑digit code, and let everyone join from their own device. Snap photos together in real time, customize layouts, apply filters/stickers, and export the final image.

## 🧠 My Contribution
- Built real‑time session sync using Socket.IO
- Integrated WebRTC + MediaPipe Selfie Segmentation for live background removal
- Designed responsive UI for cross‑device photobooth experience

## 🛠️ Tech Stack
- Frontend: Vanilla HTML, CSS, JavaScript
- Real‑Time: Socket.IO
- Camera & Segmentation: WebRTC + MediaPipe
- Backend: Node.js + Express
